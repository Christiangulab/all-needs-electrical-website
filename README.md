# All Needs Electrical Solutions

Premium Next.js 15 website for All Needs Electrical Solutions in Sydney NSW.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- App Router
- Vercel ready

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The phone number and ServiceM8 booking destination are placeholders. Update them in `lib/site.ts` and `components/ui.tsx` before launch.
